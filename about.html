History

Inspired by the ever famous Bootstrap framework, francisco was written as an exercise in understanding css and web design. Beyond that, it was created out of a newfound desire for a far more minimal framework for creating responsive web pages.

The first "complete" version was released on September 4th, 2018. Since then, the project has remained more focused on removing bloat and perfecting the code rather than adding features — however, things like unopionated buttons and common utility classes may be added in the future.
