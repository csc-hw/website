History

Inspired by the ever famous Bootstrap framework, francisco was written as an exercise in understanding css and web design. Additionally, it was created out of a desire for a far more minimal utility for creating responsive web pages.

Version 1.0 and its basic functionality were released on September 4th, 2018.
