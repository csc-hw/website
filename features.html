francisco v1.3.1 About Documentation Features

Features

Mobile minded

Media queries and fully responsive elements makes for a fluid and seamless mobile browsing experience.

Lightweight

This framework is designed to be a minimal starting point for your web projects. At just 2kb total, francisco makes a neglible impact on load speed.

Understandable

You won't need a billion and a half classes for your elements. Simple and natural tag attribution makes coding a breeze.

Specific

Only the intended page tags are affected by framework styling. It's pretty unlikely that your css will react in an unpredictable manner.

Customizable

Easily modify the framework to achieve your end goal — there's not very much css to dig through!

Reusable

Simple documentation makes it easy to apply francisco's target style with precision and consistency.
